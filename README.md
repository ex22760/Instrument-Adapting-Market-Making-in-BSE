# Instrument-Adaptive Market Making in BSE
### Combining XGBoost Regime Classification with Fractional Kelly Position Sizing

This repository contains the code and analysis scripts for the ACFIM0015 Algorithmic Trading coursework at the University of Bristol. The project develops **MMM02**, an instrument-adaptive market-making algorithm for the Bristol Stock Exchange (BSE) simulator, calibrated specifically to the intraday microstructure of the SPDR S&P 500 ETF (SPY).

---

## Overview

MMM01, the benchmark algorithm, uses fixed parameters that ignore the statistical properties of the instrument being traded. MMM02 improves on this by:

1. **Analysing SPY 1-minute training data** to identify four key microstructure properties (mean reversion, session-dependent structure, ARCH volatility clustering, fat-tailed returns)
2. **Implementing five components** motivated by those findings: FIFO inventory management, session regime switching, fractional Kelly position sizing, asymmetric sell signals, and an XGBoost buy-side classifier
3. **Running a structured ablation study** to isolate each component's contribution
4. **Validating out-of-sample** on held-out test data

## How to Run

Run the scripts in this order:

### 1. SPY Data Analysis
```bash
python3 analyse_spy.py
```
Produces four figures analysing the SPY training data microstructure. These justify the MMM02 design decisions.

### 2. Baseline Equivalence Test
```bash
python3 baseline_equivalence.py
```
Runs 25 trials with MMM02 as a verbatim clone of MMM01. Confirms no significant difference before any modifications (p=0.664), validating the experimental framework.

### 3. Single Session Balance Chart
```bash
python3 run_and_plot.py
```
Runs one 7.5-hour session and plots the accumulated balance and net worth for both traders. Change `raw_spy` to use training or test data.

### 4. Ablation Study *(takes ~2-3 hours)*
```bash
python3 ablation.py
```
Runs 25 trials for each of 6 progressive MMM02 configurations, isolating the contribution of each component. Produces the ablation figure and a results CSV.

### 5. Feature Importance Analysis *(takes ~30 minutes)*
```bash
python3 feature_importance.py
```
Runs 25 full MMM02 trials and extracts XGBoost feature importances from each calibration. Produces averaged importance plot and CSV.

### 6. Out-of-Sample Validation
Change `raw_spy` in `baseline_equivalence.py` to `spy_1m_2026-03-03.csv` and update `mm_spec` to use full MMM02 params:
```python
raw_spy = 'spy_1m_2026-03-03.csv'
mm_spec = [('MMM01', 1, {}), ('MMM02', 1, {
    'use_regime': True,
    'use_kelly': True,
    'use_asymmetric_sell': True,
    'use_xgb': True,
})]
```

## MMM02 Components

MMM02 operates in three phases: 30-minute warmup (observation only), one-off calibration at t=1800s, then active trading. Four components are controlled via boolean flags in the `params` dictionary:

| Flag | Component | Motivated by |
|---|---|---|
| always on | FIFO inventory (max 3 units) | Mean reversion (54.7% reversal rate) |
| `use_regime` | Session regime switching | Open-30min momentum vs mid-day MR |
| `use_kelly` | Fractional Kelly sizing (α=0.25) | Fat tails (kurtosis=83) |
| `use_asymmetric_sell` | Stop-loss at −8¢ | Fat-tail downside risk |
| `use_xgb` | XGBoost buy signal (P(up)>0.55) | All four microstructure findings |

## Key Results

| Configuration | Mean P&L | vs MMM01 | Cohen's d |
|---|---|---|---|
| +Inventory only | 3074¢ | +1512¢ | 0.354 |
| +Regime switching | 2759¢ | +1137¢ | 0.268 |
| +Fractional Kelly | 1975¢ | +454¢ | 0.152 |
| +Asymmetric sell | 381¢ | −23¢ | −0.029 |
| Full MMM02 (+XGB) | 537¢ | +265¢ | 0.336 |

**Key finding:** FIFO inventory management is the dominant contributor (+338¢). XGBoost feature importance analysis reveals deviation from mean (importance=0.310) as the strongest signal, while session fraction (importance=0.000) explains why regime switching degraded performance — time-of-day structure in raw data does not translate to predictive utility in BSE simulation.

**Out-of-sample:** Full MMM02 vs MMM01 on held-out test data (SPY 2026-03-03): MMM02 mean=308¢, MMM01 mean=421¢, p=0.488. No statistically significant difference, consistent with training findings. High inter-trial variance from stochastic noise traders is the primary obstacle to significance.
